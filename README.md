# みつかぶ (mitsukabu)

## 📝 特徴

- ✅ 会員登録不要・完全無料
- ✅ JPX400銘柄のローソク足チャート一覧表示
- ✅ テクニカル指標（移動平均線、RSI、MACD）
- ✅ スクリーニング機能（ターンバック、クロスV）
- ✅ 観察銘柄・検討銘柄・保有銘柄管理
- ✅ 銘柄メモ機能
